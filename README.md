This is a real-time chat application built using HTML, CSS, JavaScript, Node.js, and Socket.io. It allows multiple users to chat live with each other in a shared room. Users can enter their name, send messages, and see messages from others along with timestamps and usernames. The app demonstrates the use of WebSockets for two-way communication between the client and server.
